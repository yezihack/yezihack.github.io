## Cobra 

## 概念

Cobra有三个基本概念commands,arguments和flags。

1. commands代表行为

2. arguments代表数值

3. flags代表对行为的改变

   例:

   ```
   # clone是commands，URL是arguments，brae是flags
   git clone URL --bare
   ```
